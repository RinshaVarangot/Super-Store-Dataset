# Super-Store-Dataset

This repository contains an Advanced Superstore Sales Dashboard developed using Power BI to deliver actionable business insights from retail sales data. The dashboard provides a comprehensive analysis of key performance metrics such as total sales, profit, orders, quantity sold, customers, and products. It includes multiple analytical views such as Executive Summary, Area-wise Analysis, Customer Segmentation, Product Performance, and Ship Mode Analysis, enabling users to explore performance across different dimensions. Interactive slicers and KPI cards allow dynamic filtering and quick comparison of trends.

Data preparation was performed using Power Query to clean and transform the dataset, while DAX measures were used to create calculated metrics for deeper analysis. The dashboard highlights regional sales contributions, identifies top-performing products, and analyzes customer segment behavior. A key focus of the report is the Discount vs Profit analysis, which reveals that higher discount levels often lead to reduced or negative profitability, indicating potential inefficiencies in pricing strategies. Additionally, time-based sales and profit trends help identify fluctuations in performance, while shipping mode analysis evaluates operational efficiency.

This project demonstrates strong skills in data modeling, data cleaning, interactive dashboard design, KPI development, and business storytelling. The goal is to transform raw data into meaningful insights that support data-driven decision-making and improve overall business performance. 📊🚀
